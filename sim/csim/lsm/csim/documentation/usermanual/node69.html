GVD_cT_Gate

Read/writable Fields

Vh (Volt) :
Inflection point of the sigmoidal function p(V)
Vc (Volt) :
Slope of the sigmoidal function p(V)
Ts :
Voltage independent time constant
nummethod (flag) :
Numerical method for the solution of the differential equation: Exp. Euler = 0, Crank-Nicolson = 1
k (1) :
The exponent of the gate
p :
The state variable.

Readonly Fields

P (1) :
The output P(t,V) = p(t,V)k of the gate.
