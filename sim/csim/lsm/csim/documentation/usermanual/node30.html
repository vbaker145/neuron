CSIM command reference

The common form of all CSIM commands is 

  csim(command,...);
where command is one of the following strings:

  • 'create' : Create an object of a certain class
  • 'set' : Set fields of an object
  • 'connect' : Connect objects
  • 'get' : Get field values of an object
  • 'reset' : Reset the simulation to t=0.0
  • 'simulate' : Simulate the network
  • 'export' : Export the network for saving it
  • 'import' : Import a loaded network
  • 'destroy' : Destroy/delete the current netork
  • 'list' or 'ls' List various items
  • 'version' : Print out a version string
